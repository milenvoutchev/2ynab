## Source
The bank2ynab.conf file is taken directly from the https://github.com/bank2ynab/ project, which is an open-source tool for converting bank transaction exports into a format compatible with YNAB (You Need A Budget).
The file is left intentionally unmodified so that it can be copy-pasted into our project without any discrepancies. 

SOURCE: https://github.com/bank2ynab/bank2ynab/blob/develop/bank2ynab/data/bank2ynab.conf
